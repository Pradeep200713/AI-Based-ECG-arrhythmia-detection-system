# AI-Based ECG Arrhythmia Detection System

## 📌 Overview
This project is a Machine Learning-based system that analyzes ECG signals to detect abnormal heart rhythms (arrhythmia). It classifies ECG signals as **Normal** or **Abnormal**, helping in early diagnosis of heart conditions.

## 🎯 Objective
- Detect arrhythmia using ECG signals  
- Build an accurate ML model  
- Assist in early diagnosis of heart diseases  

## 🧠 Technologies Used
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib  
- Flask

## 📊 Dataset
- MIT-BIH Arrhythmia Dataset  

## ⚙️ How to Run

### Step 1: Install dependencies
pip install -r requirements.txt

### Step 2: Train the model
python train.py

### Step 3: Run the app
python app.py
